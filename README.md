# RESTAURANT-SALES-ANALYSIS-IN-PYTHON

## Overview
This project analyzes customer purchase data using Python to uncover sales trends and customer spending patterns. The analysis focuses on revenue performance across different months and customer demographics.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Objectives
- Analyze monthly sales performance
- Compare revenue generated by different genders
- Identify purchasing trends and patterns
- Create visualizations to support business insights

## Key Findings
- January recorded the highest sales revenue.
- December was the second-best performing month.
- February had the lowest sales among the analyzed months.
- Male customers generated more total revenue than female customers.

## Project Workflow
1. Data Cleaning and Preparation
2. Exploratory Data Analysis (EDA)
3. Revenue Aggregation
4. Data Visualization
5. Insight Generation

## Visualizations
- Monthly Sales Trend
- Gender Revenue Comparison

## Business Insights
The analysis highlights sales fluctuations across months and differences in customer spending behavior. These insights can support better forecasting, inventory management, and targeted marketing strategies.
